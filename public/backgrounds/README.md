# Фоны для PDF семейного древа

Готовые фоновые изображения для встроенных пресетов.
Кладутся сюда и раздаются статически через `/backgrounds/...`.

## Форматы файлов

| Формат страницы | Пропорция | Размер px (рекомендуемый) | Имя файла |
|---|---|---|---|
| A4 Landscape | 842×595 pt → **1.415:1** | **2480×1754** | `*_a4l.jpg` |
| A4 Portrait | 595×842 pt → **1:1.415** | **1754×2480** | `*_a4p.jpg` |
| A3 Landscape | 1191×842 pt → **1.414:1** | **3508×2480** | `*_a3l.jpg` |
| A3 Portrait | 842×1191 pt → **1:1.414** | **2480×3508** | `*_a3p.jpg` |

> Разрешение 300 DPI. Можно PNG (если нужна прозрачность) или JPEG (меньше вес).

## Стили (по 4 формата каждый = 4 файла)

### 1. `vintage` — Пергамент / старая бумага
- Тёплый бежевый (#f7f0d9 → #e8e0c8), текстура состаренной бумаги
- Тонкая декоративная рамка по периметру (коричневая)
- Лёгкие виньетки по углам
- **Файлы:** `vintage_a4l.jpg`, `vintage_a4p.jpg`, `vintage_a3l.jpg`, `vintage_a3p.jpg`

### 2. `tree` — Дерево жизни
- Центральный ствол дерева снизу, ветви расходятся к углам
- Листочки по краям (зелёные, разных оттенков)
- Фон: светло-бежевый/кремовый
- Двойная золотисто-коричневая рамка
- **Файлы:** `tree_a4l.jpg`, `tree_a4p.jpg`, `tree_a3l.jpg`, `tree_a3p.jpg`

### 3. `modern` — Чистый минимализм
- Белый/светло-серый (#ffffff → #f3f4f6)
- Тонкие геометрические линии или точки по углам
- Без рамки или очень тонкая (#e5e7eb)
- **Файлы:** `modern_a4l.jpg`, `modern_a4p.jpg`, `modern_a3l.jpg`, `modern_a3p.jpg`

### 4. `dark` — Тёмная элегантность
- Тёмно-синий/графитовый (#0f172a → #1e293b)
- Тонкие золотые или серебряные акценты по углам
- Лёгкий градиент или текстура ткани
- **Файлы:** `dark_a4l.jpg`, `dark_a4p.jpg`, `dark_a3l.jpg`, `dark_a3p.jpg`

### 5. `sakura` — Цветение сакуры
- Нежно-розовый (#fff1f5 → #ffe4ec)
- Ветки сакуры с лепестками по углам
- Лёгкая акварельная текстура
- **Файлы:** `sakura_a4l.jpg`, `sakura_a4p.jpg`, `sakura_a3l.jpg`, `sakura_a3p.jpg`

### 6. `forest` — Лесная зелень
- Светло-зелёный (#f0f8f0 → #d9ead3)
- Листья/папоротники по краям
- Тонкая зелёная рамка
- **Файлы:** `forest_a4l.jpg`, `forest_a4p.jpg`, `forest_a3l.jpg`, `forest_a3p.jpg`

### 7. `gold` — Золотая классика
- Кремовый фон с золотым орнаментом по периметру
- Уголки с завитками
- Центр чистый (для карточек)
- **Файлы:** `gold_a4l.jpg`, `gold_a4p.jpg`, `gold_a3l.jpg`, `gold_a3p.jpg`

### 8. `watercolor` — Акварель
- Мягкие пастельные разводы по краям (голубой/лавандовый/персиковый)
- Центр почти белый
- Без рамки
- **Файлы:** `watercolor_a4l.jpg`, `watercolor_a4p.jpg`, `watercolor_a3l.jpg`, `watercolor_a3p.jpg`

## Итого: 8 стилей × 4 формата = 32 файла

## Требования к изображениям

1. **Центр оставить свободным** — там будут карточки (примерно 60-70% площади)
2. **Декор по краям** — рамки, ветки, орнаменты, виньетки
3. **Цвета приглушённые** — фон не должен конкурировать с текстом карточек
4. **Формат:** JPEG quality 90% (или PNG если нужна прозрачность)
5. **Именование:** `{style}_{format}.jpg` — например `vintage_a4l.jpg`
